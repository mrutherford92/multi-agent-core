# Standing Order 010: Issue Hygiene

**From Captain Michael. ALL AGENTS.**

## Rules
1. Issues are for ACTIONABLE WORK only — must have a deliverable and acceptance criteria.
2. Status updates are NOT issues — comment on the parent issue or update your status file.
3. Handoff issues must contain concrete next-step work, not notifications.
4. Bridge audits open issues weekly and closes stale/noise issues.

Violation = immediate close with process flag.
