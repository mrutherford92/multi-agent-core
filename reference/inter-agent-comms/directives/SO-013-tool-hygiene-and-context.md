# Standing Order 013: Tool Hygiene & Context Awareness

**From Captain Michael.**

## Mandatory Protocols

1. **Verify Your Workspace:** Before running heavy search commands or executing file modifications, ALWAYS confirm you are operating in the correct repository context. Do not search outside of your assigned territory unless explicitly directed.
2. **Ask Before Thrashing:** If a task references files, components, or specific UI text that cannot be quickly located in the current codebase, **STOP AND ASK**. Do NOT run rampant, recursive brute-force searches if initial targeted searches fail. 
3. **Acknowledge Platform Boundaries:** If prompt logic seems completely alien to the current tech stack, immediately halt and ask the Captain if the prompt was issued to the correct agent.
4. **Abort on Stalls:** If a tool hangs or stalls, terminate it immediately. Do not queue up repeated attempts. Re-evaluate your approach and ask for guidance.
